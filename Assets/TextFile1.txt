
Mathf!

- Mathf.clamp = Clamps a value between a minimum float and maximum float value.
// Clamps the value 10 to be between 1 and 3.
// prints 3 to the console

Debug.Log(Mathf.Clamp(10, 1, 3));

- Mathf.Sign(); Return value is 1 when f is positive or zero, -1 when f is negative.
Debug.Log(Mathf.Sign(-10));
Debug.Log(Mathf.Sign(10));

- Mathf.Abs(value); Returns the absolute value of f.

// prints 10.5
Debug.Log(Mathf.Abs(-10.5));

- Mathf.SmoothDamp(float current, float target, ref float currentVelocity, float smoothTime, float maxSpeed = Mathf.Infinity, float deltaTime = Time.deltaTime);
Gradually changes a value towards a desired goal over time.
https://docs.unity3d.com/ScriptReference/Mathf.SmoothDamp.html
The value is smoothed by some spring-damper like function, which will never overshoot. The function can be used to smooth any kind of value, positions, colors, scalars.
